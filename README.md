## How to run tutorials

### System Requirements
* jdk version 17 or later
* docker-compose
* node.js version 14 or later
* npm version 8 or later

### Build tutorial sample application
```
sh build_and_run_tutorial.sh
```

### See tutorial
http://localhost:7007

### OpenAPI
http://localhost:7007/swagger-ui.html

## Run test (jest)
```
cd tutorial+test
npm install
npm run test
```
